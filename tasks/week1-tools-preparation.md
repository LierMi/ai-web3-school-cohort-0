# Week 1 Tools Preparation｜基础工具准备记录

## 概览

这份文档记录我为 Week 1 学习和实践任务准备的基础工具。我的策略不是一次性安装所有工具，而是先准备一套轻量、够用、可持续扩展的工具组合，用于协作沟通、AI 辅助开发和 Web3 基础实践。

本提交不包含任何私钥、助记词、API Key、token、`.env` 文件或其他敏感信息。

## 4. 这些工具如何支持 Week 1

Week 1 的重点是理解 Web3 基础开发流程、建立安全的测试环境，并形成稳定的协作节奏。

协作工具用于跟进课程安排、接收通知、参与讨论和管理截止时间。AI 工具用于帮助我更快理解技术概念、整理项目想法、生成 README 和 Demo Story。Web3 工具用于练习钱包使用、交易查看、测试网交互和基础合约操作，同时避免暴露真实资产。

## 5. 安全注意事项

- 不提交私钥、助记词、API Key、token、`.env` 文件或钱包恢复信息。
- 使用单独的测试钱包完成学习和黑客松相关实验。
- 钱包插件、区块浏览器、开发工具等尽量从官方入口访问。
- 不随意连接陌生网站，不签署看不懂的消息或交易。

## 6. 尚未完成的准备与下一步计划

- 确认课程指定沟通入口，并把所有重要截止时间加入 Google Calendar。
- 如果后续需要更频繁地修改代码，完成 Cursor 的安装和配置。
- 在钱包中添加 Sepolia 或其他 EVM 测试网，并从官方 faucet 获取测试币。
- 练习使用区块浏览器查看交易、地址和合约信息。
- 使用 Remix 尝试一次简单的合约部署或交互。
- 如果项目进入智能合约开发阶段，再安装 Hardhat 或 Foundry。
