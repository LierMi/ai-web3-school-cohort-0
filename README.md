## 关于 AI × Web3 School

AI × Web3 School 是一个面向 AI × Web3 交叉方向的共学与实践项目，不是概念课程，而是从真实约束出发，帮助参与者建立共同语言、完成基础实践、将想法转化为可演示、可验证、可延续的项目。

**完整路径**：问题定义 → Bootcamp 训练 → 动手实践 → 项目开发 → 公开展示 → 人才与机会积累

- 官网：https://aiweb3.school
- Handbook：https://aiweb3.school/zh/handbook/
- 课程平台：https://web3career.build/zh/programs/AI-Web3-School

> 隐私提醒：本仓库为公开仓库，请勿提交私钥、助记词、API Key、.env 文件或任何敏感信息。
